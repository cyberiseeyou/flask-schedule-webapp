What is the Scheduler?

The Product Connections Scheduler is an automated scheduling system designed to efficiently assign field events to your team of employees. It integrates with Walmart's Retail Link system to manage Core events, Digitals, Juicers, and Supervisor events.

Key Features

  • Automatic Event Synchronization: Pulls unscheduled events from Walmart Retail Link automatically
  • Intelligent Scheduling: Assigns events based on employee skills, availability, and workload
  • Rotation Management: Manages daily rotations for Juicers and Primary Leads
  • Conflict Detection: Prevents scheduling conflicts and respects time-off requests
  • Review Workflow: Allows managers to review and adjust schedules before submission
  • API Integration: Automatically submits approved schedules back to Walmart's system

Navigation Overview

The main navigation bar provides access to all key features:

  • Dashboard: Overview of scheduling status and pending actions
  • Events: View and manage unscheduled events
  • Calendar: Visual calendar view of scheduled events
  • Employee Management: Add and manage team members
  • Analytics: Employee performance and scheduling statistics
  • Printing: Print schedules and reports
  • Settings (in user dropdown): Configure system settings and credentials
