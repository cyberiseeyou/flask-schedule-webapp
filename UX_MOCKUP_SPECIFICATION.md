## Executive Summary

This specification details the complete UI/UX redesign of the calendar system, transforming it from a constrained modal-based interface to a full-screen, information-dense, and user-friendly scheduling dashboard.

### Key Improvements:
- **50% more screen real estate** on desktop (1200px â†’ full viewport)
- **At-a-glance event overview** via color-coded badges
- **Proactive alerts** for unscheduled events
- **Dedicated daily view** replacing cramped modals
- **Transparent CORE-Supervisor pairing** with inline management

---

## Design Principles

1. **Information Density Without Clutter** - Show more data, but organize it clearly
2. **Progressive Disclosure** - Start with overview, drill down as needed
3. **Consistent Action Patterns** - Same buttons work the same way everywhere
4. **Visual Hierarchy** - Color, size, and position guide the eye
5. **Responsive by Default** - Mobile-first thinking, desktop-optimized execution

```javascript
  'button, input, select, [tabindex]:not([tabindex="-1"])'

      e.preventDefault();
      firstElement.focus();

    closeModal();
  }
});

// Focus first element on open
firstElement.focus();
```

```python
    if not supervisor_event:
        return {
            "status": "not_found",
            "message": "No Supervisor event configured"
        }

    if supervisor_event.condition == 'Scheduled':
        schedule = Schedule.query.filter_by(
            event_id=supervisor_event.event_id
        ).first()

        return {
            "status": "scheduled",
            "employee": schedule.employee.name,
            "time": schedule.start_datetime.strftime('%I:%M %p'),
            "event_id": supervisor_event.event_id
        }
    else:
        return {
            "status": "unscheduled",
            "message": "Supervisor Event NOT Scheduled",
            "event_id": supervisor_event.event_id
        }
```

### 3. Reschedule/Unschedule Pairing

**Reschedule Logic:**
```python
@app.route('/api/reschedule_event', methods=['POST'])
def reschedule_event():
    schedule_id = request.json['schedule_id']
    new_date = request.json['new_date']
    new_employee_id = request.json.get('new_employee_id')

    schedule = Schedule.query.get(schedule_id)
    event = Event.query.get(schedule.event_id)

    # Update CORE event schedule
    schedule.start_datetime = new_date
    if new_employee_id:
        schedule.employee_id = new_employee_id

    # If this is a CORE event, also reschedule supervisor
    if '-CORE-' in event.project_name:
        supervisor_event = get_supervisor_event(event)
        if supervisor_event:
            supervisor_schedule = Schedule.query.filter_by(
                event_id=supervisor_event.event_id
            ).first()

            if supervisor_schedule:
                # Reschedule supervisor to same date at 12:00 PM
                supervisor_date = datetime.combine(
                    new_date.date(),
                    datetime.strptime('12:00', '%H:%M').time()
                )
                supervisor_schedule.start_datetime = supervisor_date

    db.session.commit()
    return jsonify({"success": True})
```

**Unschedule Logic:**
```python
@app.route('/api/unschedule/<int:schedule_id>', methods=['DELETE'])
def unschedule(schedule_id):
    schedule = Schedule.query.get(schedule_id)
    event = Event.query.get(schedule.event_id)

    # Unschedule CORE event
    event.condition = 'Unstaffed'
    event.is_scheduled = False
    db.session.delete(schedule)

    # If this is a CORE event, also unschedule supervisor
    if '-CORE-' in event.project_name:
        supervisor_event = get_supervisor_event(event)
        if supervisor_event:
            supervisor_schedule = Schedule.query.filter_by(
                event_id=supervisor_event.event_id
            ).first()

            if supervisor_schedule:
                supervisor_event.condition = 'Unstaffed'
                supervisor_event.is_scheduled = False
                db.session.delete(supervisor_schedule)

    db.session.commit()
    return jsonify({"success": True})
```
